# 💳 Credit Card Financial Dashboard | Power BI + SQL

An interactive Power BI dashboard connected to a SQL database for analyzing credit card business performance, customer behavior, transactions, and revenue through weekly business intelligence reporting.

---

## 🗂️ Project Overview

| Detail | Info |
|---|---|
| Tool | Power BI Desktop |
| Data Source | SQL Database |
| Records | 10,293+ |
| Dashboard Pages | 3 |
| Revenue | $56.5M |
| Interest Earned | $7.98M |
| Transactions | 667K |
| Technologies | Power BI, SQL, DAX, Power Query |

## 🚀 How to Run

1. Import the provided CSV files into your SQL database.
2. Execute `ccdb_queries.sql`.
3. Open the `.pbix` file in Power BI Desktop.
4. Update the SQL connection if required.
5. Refresh the report.
